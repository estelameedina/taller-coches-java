# Aplicación de Gestión de Taller de Coches

Aplicación desarrollada en Java como parte de mi formación en Desarrollo de Aplicaciones Web (DAW).  

El proyecto simula la gestión básica de un taller de coches, permitiendo organizar clientes, vehículos y reparaciones mediante programación orientada a objetos.

---

## Tecnologías utilizadas
- Java  
- Programación Orientada a Objetos (POO)  

---

## Funcionalidades
- Gestión de clientes  
- Gestión de vehículos  
- Registro de reparaciones  
- Asociación de vehículos a clientes  
- Almacenamiento de datos mediante colecciones  
- Manejo de errores con excepciones personalizadas  

---

## Conceptos aplicados
- Diseño de clases y objetos  
- Encapsulación  
- Uso de colecciones (listas)  
- Separación de responsabilidades  
- Control de errores y validaciones  

---

## Objetivo del proyecto
Desarrollar una aplicación completa aplicando los principios de la programación orientada a objetos y la gestión de datos en Java.

---

## Ejecución
1. Clonar el repositorio  
2. Abrir el proyecto en un IDE (IntelliJ, Eclipse, etc.)  
3. Ejecutar la clase principal (`Main`)  

---

## Ejemplo de uso

El programa permite:
- Crear coches
- Añadirlos al taller
- Gestionar reparaciones

La ejecución se realiza desde consola mediante la clase `Main`.
